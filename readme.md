## ðŸ§® RFM Methodology

RFM scoring is calculated based on:
- **Recency**: Days since last purchase
- **Frequency**: Number of orders
- **Monetary**: Total profit

Customers are ranked and segmented into:
- **Champions** (Top 10%)
- **Loyal Customers** (Next 15%)
- **Potential Loyalists**
- **At Risk**
- **Hibernating** (Bottom 25%)
